這邊用來存放一些跟新文章時需要用的指令
希望我能週更我的 blog : )

新增文章

  • 如果標題有空格，需加入" "
  • markdown 語法參考、另一個

刪除文章

  1. 直接在 source/_post 資料夾中刪除文章。
  2. 在終端機執行 hexo clean，並在根目錄下刪除 db.json。
  3. 執行 hexo generate 或是 hexo g。
  4. 執行 hexo deploy 或是 hexo d 重新發佈。

clean 清除靜態檔案與快取

在每次儲存修正後的檔案之前，會建議先輸入此指令，清除快取檔案 （db.json）和已產生的靜態檔案（public）。
