Intro

終於來到最後一篇 🫠
最後壓軸登場的是 Transition，可以再細分成 2D 和 3D，走起介紹囉！

2D Transition 屬性

  • translate() 位移
  • rotate() 旋轉
  • scale() 縮放
  • skew() 傾斜
  • matrix() 依變形矩陣的 6 個參數值產生 2D 變形

2D Transition 語法＆示範

translate

  • 單位可以用px或%
  • 如果只寫一個參數，省略的第 2 個參數會視為 0，也就是只有 X 軸的移動距離
  • 可細分為 X 軸與 Y 軸的位移，屬性如下：translateX() 、 translateY()

rotate

  • 單位用deg，正值=順時針旋轉，負值=逆時針旋轉
  • 沒有指定基準點就是以中心點為基準

scale

  • 單位無，為倍數值
  • 沒有指定基準點就是以中心點為基準

skew

  • 單位用deg
  • 沒有指定基準點就是以中心點為基準
  • 可細分為 X 軸與 Y 軸的縮放，屬性如下：skewX()、skewY()

matrix

結合以上屬性來做 2D Transform
matrix(a, b, c, d, e, f);
參數 a 和 d：控制元素的縮放
參數 b 和 c：控制元素的傾斜或斜切
參數 e 和 f：控制元素的平移
